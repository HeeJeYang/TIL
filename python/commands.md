## Python Commands For Me

- insert(0, 0) : 리스트의 앞쪽에 원소 추가
- pop(0) : 리스트의 0번째 인덱스 값 삭제
- append() : 리스트 뒤에 새 원소 추가
- pop() : 리스트 마지막 인덱스 값 삭제
- ord() : 유니코드 또는 알파벳 문자의 ASCII 값 반환
- "<구분자>".join(<리스트>) : 리스트를 문자열로 일정하게 합쳐줌(구분자로 구분)
- <문자열>.upper() : 모든 알파벳들을 대문자로 변환 ('abc' -> 'ABC')
- <문자열>.captialize() : 문자열의 첫 글자를 대문자로 변환'abc' -> 'Abc'
- <문자열>.title() : 문자열에서 알파벳 외의 문자(숫자, 특수기호, 띄어쓰기 등)로 나누어져 있는 영단어들의 첫 글자들을 모두 대문자로 변환
- max(<리스트>) : 리스트 내 최댓값 반환
- min(<리스트>) : 리스트 내 최솟값 반환
- round(a, b) : a를 b째 자리까지 반올림
- sum(<리스트>) : 리스트 내 원소들의 합
